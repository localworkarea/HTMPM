Лабораторія електронної мікроскопіїї

Містить у своєму складі електронний мікроскоп РЕМ106І, яки й дозволяє досліджувати мікроструктуру на макро- та мікрорівні при збільшеннях аж до 10 000. Мікроскоп обладнаний приставкою Oxsford для проведення мікрорентгеноспектрального аналізу

Детальніше

Лабораторія рентгено-фазового аналізу

Використовує дифрактометр Ultima IV, виробництва японської фірми Rigaku, для визначення фазового складу матеріалу та наявних в ньому залишкових напружень

Детальніше

Лабораторія підготовки шліфів

Призначена для підготовки зразків до металографічних, механічних, ренгенівських досліджень тощо. Матеріали проходятьдекілька етапів підготовки в залежності від вимог методу вимірювання

Детальніше

Лабораторія механічних випробувань

Має у своєму складі установку для вимірювання міцності зразків на розтяг, стиснення, твердості за різними шкалами

Детальніше

Лабораторія НАТО

Установка іскроплазмового спікання використовує одночасний вплив температури та тиску для консолідації суміші керамічних або металокерамічних порокшів

Детальніше

Лабораторія аналізу порошків

Містить у своємо складі сучасне обладнання для подрібнення і вимірювання розмірів порошків

Детальніше

Лабораторія 3D-моделювання

Використовує сучасі програми аналізу властивостей матеріалів. Компютери мають спеціальне програмне забезпечення для створення 3D моделі виробів, а також відтворення умов їх роботи у збірці

Детальніше

Лабораторія високотемпературних досліджень

Використовується для вимірювання фазових перетворень, що відбуваються під час нагрівання матеріалів

Детальніше

Лабораторія оптичної мікроскопії

Використовує світлові мікроскопи для дослідження розмірів зерен та пористості матеріалів
